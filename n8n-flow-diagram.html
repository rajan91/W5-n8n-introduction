03 — Deployment

Platform Setup Types

Local Setup Self-Host n8n Production n8n Cloud

💻 Local Setup

  • npm install n8n -g
  • Node.js v18+ required
  • localhost:5678
  • Dev & testing only
  • SQLite (default)
  • No SSL / No auth

🐳 Self-Host

  • Docker / Kubernetes
  • VPS / Cloud VM
  • Full data control
  • Custom config
  • Manage updates yourself
  • Private integrations

🚀 n8n Production

  • PostgreSQL DB
  • Docker Compose
  • Nginx + SSL/TLS
  • Basic Auth / SSO
  • Backups + Monitoring
  • Restart policies
